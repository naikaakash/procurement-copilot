# 🚀 Deployment Notes

*Guidelines for local deployment and staging servers.*

### 💻 Local Run
1. Run `npm install` to acquire Node modules.
2. Sourced files from `/procurement_data_sample` folder.
3. Start the Turbopack development server via `npm run dev`.
4. Access dashboard via `http://localhost:3000`.

### ☁️ Cloud Stage (Phase 1B Scope)
* Standardized Next.js Node deployment on Vercel or Azure App Service.
* PostgreSQL database connection strings established in environment variables.
