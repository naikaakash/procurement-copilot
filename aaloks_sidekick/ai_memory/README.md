# 🧠 AI Memory Folder

This folder serves as the central brain of Aalok's Sidekick. It keeps a record of all engineering experience, design patterns, prompts, journals, and bug logs.

* **`app_build_journal.md`**: Chronological log of build decisions.
* **`lessons_learned.md`**: Core operational mistakes and wisdom.
* **`reusable_patterns.md`**: Technical blocks (e.g., service layers).
* **`prompt_library.md`**: Prompts that yielded high-quality code.
* **`starter_templates.md`**: Copy-paste starters.
* **`architecture_decisions.md`**: ADR style records.
* **`bug_fixes.md`**: Specific exceptions and how they were resolved.
* **`ui_preferences.md`**: Fonts, variables, drawer styles.
* **`deployment_notes.md`**: Cloud and local hosting instructions.
* **`phase_end_summaries.md`**: Summaries of curated milestones.
* **`reusable_components.md`**: Premium UI blocks.
* **`future_project_bootstrap.md`**: Active blueprints for starting new codebases.
