# 🤖 Dev Agents Folder

This folder defines standard persona scripts and execution instructions for the development helper agents.

* **`product_manager_agent.md`**: Keeps focus on phase and stops scope creep.
* **`solution_architect_agent.md`**: Manages modular designs.
* **`frontend_developer_agent.md`**: Enforces sleek CSS dark theme variables.
* **`backend_developer_agent.md`**: Guides routing and API performance.
* **`data_layer_agent.md`**: Decoupled service layer verification.
* **`ui_ux_agent.md`**: Review contrast, layout, fixed drawers.
* **`qa_agent.md`**: Controls testing checklists.
* **`ba_domain_agent.md`**: Validates SAP procurement flow usefulness.
* **`release_manager_agent.md`**: Version checks and deployment scripts.
* **`memory_curator_agent.md`**: Automates memory capture when curating phases.
* **`future_project_bootstrap_agent.md`**: Handles auto-bootstrap rules.
