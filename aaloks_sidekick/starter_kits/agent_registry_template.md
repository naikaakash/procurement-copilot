# 🤖 Agent Registry Template

Standard list of development agents to equip in your new codebases:

1. **PM Agent**: Restricts scope, controls phase milestones.
2. **BA Domain Agent**: Checks procurement calculations (Overdue Days).
3. **UI/UX Agent**: Enforces Dark Slate variable consistency.
4. **QA Agent**: Mandates build compilation checks.
5. **Memory Curator**: Automates phase-end learnings storage.
